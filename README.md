# 🍓 10 D Cafe – Fruit Salad Website

A premium mobile-first website for **10 D Cafe** – a fresh fruit salad shop.

## 📁 Files
- `index.html` – Main website (single file, no dependencies)

## 🚀 How to Run on GitHub Pages (Live Tab)

1. **Create a GitHub account** at [github.com](https://github.com)
2. Click **"New Repository"** → Name it `10dcafe` → Set to **Public** → Click Create
3. Click **"uploading an existing file"** → Drag & drop `index.html`
4. Commit the file
5. Go to **Settings → Pages → Source → main branch** → Save
6. Your live site will be at: `https://YOUR-USERNAME.github.io/10dcafe`
